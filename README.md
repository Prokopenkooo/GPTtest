# GPTtest

Тестовый репозиторий для работы с Codex.

## Цель

Репозиторий используется как лаборатория для проверки идей, написания кода и подготовки будущих программных решений ПРОЛЕКС.

## Направления

- утилиты для инженерных задач;
- автоматизация процессов;
- прототипы интеграций;
- подготовка к разработке решений для КОМПАС-3D, ЛОЦМАН PLM, ВЕРТИКАЛЬ и ПОЛИНОМ MDM.
